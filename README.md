# Game-DIstribution-Website
## How to install
```
git clone https://github.com/andrewquang512/Game-DIstribution-Website.git
```
## Install necessary packages
### Install packages of client
```
cd client
npm install
```
### Install packages of backend
```
cd backend
npm install
```
## How to run
### Run backend
```
cd backend
npm run server
```
### Run client
```
cd client
npm start
```
